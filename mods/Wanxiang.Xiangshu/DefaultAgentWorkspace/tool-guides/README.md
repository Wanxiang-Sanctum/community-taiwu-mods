# 运行工具指引入口

你是读取这些文件的 Agent。`tool-guides/` 存放相枢默认工作区的运行工具使用指引，用来帮助你高效使用
当前可用工具。玩家可见答复由你根据工具结果另行组织。

本目录随相枢 Mod 发布，必须保持自包含。这里的内容提供默认相枢运行工具背景、常见脚本入口和游戏知识
检索路线，用来辅助理解当前可用工具说明；工具是否存在、参数名称、目标侧和副作用以当前可用工具说明为准。

## 读取路由

| 当前请求需要 | 读取文件 | 使用方式 |
| --- | --- | --- |
| 需要查询游戏机制、术语、配置、本地化、模板/显示辅助、百晓册/百科文本，或需要先找到游戏内名称和说明 | `GAME_KNOWLEDGE.md` | 用来选择直接知识入口，优先查配置、本地化、模板辅助和百晓册，减少反射探索。 |
| 需要读取或改变当前游戏、存档、界面或 mod 运行状态，并且当前可用工具包含脚本/工具能力 | `RUNTIME_SCRIPTING.md` | 用来选择目标侧、组织最小脚本和理解工具结果。 |

普通闲谈、静态世界观解释、写作请求和已经有足够事实的回合直接回答；本目录用于工具相关请求。

## 模块边界

- `README.md`：说明何时读取运行工具指引，以及这些指引和实际工具说明的关系。
- `GAME_KNOWLEDGE.md`：说明游戏机制、配置、本地化、模板/显示辅助和百晓册资料的直接检索入口，以及反射探索边界。
- `RUNTIME_SCRIPTING.md`：说明默认相枢脚本工具的使用方式、前后端运行环境差异、入口契约和常见运行时锚点。

本目录只提供运行工具使用指引。玩家可见人设归 `persona/`，世界观资料归 `lore/`，运行期记录归
`.xiangshu-runtime/`，源码维护流程归 `docs/`。
