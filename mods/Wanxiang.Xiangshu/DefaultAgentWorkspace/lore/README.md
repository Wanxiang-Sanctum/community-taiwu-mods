# 世界观资料入口

你是读取这些文件的 Agent。`lore/` 存放 Agent 工作区内的静态世界观资料，用来辅助你生成玩家可见答复；
玩家可见答复由你按当前请求组织。

本目录提供相枢可读取的静态世界观资料。静态世界观以本目录为准。当前角色、地点、界面、
背包、任务阶段、实时数值和运行故障也不在这里，以当前输入或可用工具取得的事实为准。

## 读取路由

| 当前请求需要                                                                             | 读取文件            | 使用方式                                         |
| ---------------------------------------------------------------------------------------- | ------------------- | ------------------------------------------------ |
| 解释太吾、伏虞剑柄、剑冢、相枢爪牙、入邪、入魔、传承等基础概念                           | `WORLD_BASELINE.md` | 给基础、可直接理解的说明。                       |
| `WORLD_BASELINE.md` 不足以回答相枢本体、化身、侵袭进度、剑冢结构、魔血、焕心、三魔等问题 | `XIANGSHU.md`       | 先取开头的答复锚点；问题确实需要时再读后段专题。 |

## 模块边界

- `WORLD_BASELINE.md`：告诉你如何解释基础概念。
- `XIANGSHU.md`：告诉你如何处理基础概念之外的相枢专题资料。

本目录只提供答复时需要的静态世界观资料。基础人设口吻按 `AGENTS.md` 处理，更细的人设校准读取 `persona/`；
当前事实和运行状态以当前输入或可用工具取得的结果为准。
