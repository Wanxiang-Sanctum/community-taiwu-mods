# 世界观资料入口

需要静态太吾世界观摘要时读取本目录。只把这里当作随默认工作区发布的背景和回答锚点；不要把它当作完整游戏
资料库。当前角色、地点、界面、背包、任务阶段、实时数值、运行故障和当前世界进度，以当前输入或可用工具
取得的事实为准。

## 读取路由

| 当前请求需要                                                                             | 读取文件            | 使用方式                                         |
| ---------------------------------------------------------------------------------------- | ------------------- | ------------------------------------------------ |
| 解释太吾、伏虞剑柄、剑冢、相枢爪牙、入邪、入魔、传承等基础概念                           | `WORLD_BASELINE.md` | 给基础、可直接理解的说明。                       |
| `WORLD_BASELINE.md` 不足以回答相枢本体、化身、侵袭进度、剑冢结构、魔血、焕心、三魔等问题 | `XIANGSHU.md`       | 先取开头的答复锚点；问题确实需要时再读后段专题。 |

## 使用边界

- 只从本目录取静态世界观资料和回答锚点。基础人设口吻按 `AGENTS.md` 处理，更细的人设校准读取 `persona/`。
- 玩家问当前状态时，不从静态资料推断当前事实；按 `AGENTS.md` 和 `tool-guides/README.md` 取得运行事实。
- 深层主线、相枢真身、焕心、三魔等内容只在玩家明确追问或接受剧透时展开。

## 叶子文件

- `WORLD_BASELINE.md`：取基础概念、普通机制解释和需要当前事实的问题边界。
- `XIANGSHU.md`：取基础概念之外的相枢专题资料、侵袭阶段、剑冢结构、魔血意象和主线真相素材。
