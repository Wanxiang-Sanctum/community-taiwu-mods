# Agent 能力上下文入口

你是读取这些文件的 Agent。`capabilities/` 存放相枢默认工作区的运行能力上下文，用来帮助你高效使用本轮
实际可用的工具；不要把这些文件当成要展示给玩家的说明书。

本目录随相枢 Mod 发布，必须保持自包含。这里的内容提供默认相枢运行能力背景、常见脚本入口和游戏知识
检索路线，用来辅助理解本轮实际工具说明；工具是否存在、参数名称和副作用以本轮实际工具说明为准。

## 读取路由

| 本轮需要 | 读取文件 | 使用方式 |
| --- | --- | --- |
| 需要查询游戏机制、术语、配置、本地化、模板/显示辅助、百晓册/百科文本，或需要先找到游戏内名称和说明 | `KNOWLEDGE.md` | 用来选择直接知识入口，优先查配置、本地化、模板辅助和百晓册，减少反射探索。 |
| 需要读取或改变当前游戏、存档、界面或 mod 运行状态，并且本轮暴露了脚本/工具能力 | `SCRIPTING.md` | 用来选择目标侧、组织最小脚本和理解工具结果。 |

普通闲谈、静态世界观解释、写作请求和已经有足够事实的回合不需要读取本目录。

## 模块边界

- `README.md`：说明何时读取能力上下文，以及能力上下文和实际工具说明的关系。
- `KNOWLEDGE.md`：说明游戏机制、配置、本地化、模板/显示辅助和百晓册资料的直接检索入口，以及反射探索边界。
- `SCRIPTING.md`：说明默认相枢脚本工具的使用方式、前后端运行环境差异、入口契约和常见运行时锚点。

本目录只提供运行能力使用上下文，不承载玩家可见人设、世界观资料、运行期记录、诊断日志或源码维护流程。
