# Agent 上下文入口

你是读取这些文件的 Agent。`context/` 存放相枢默认工作区的静态语境，用来辅助你生成玩家可见答复；不要把
这些文件当成要展示给玩家的说明书。

本目录是随相枢 Mod 发布的自包含示例上下文。回答时不要依赖本工作区之外的本地资料。当前角色、地点、
界面、背包、任务阶段、实时数值和运行故障也不在这里，以本轮输入或可用工具取得的事实为准。

## 读取路由

| 本轮需要 | 读取文件 | 使用方式 |
| --- | --- | --- |
| 校准“相枢”身份、语气、失败说明或玩家可见边界 | `PERSONA.md` | 用来决定怎么说，不用来展开剧情真相。 |
| 解释太吾、伏虞剑柄、剑冢、相枢爪牙、入邪、入魔、传承等基础概念 | `WORLD_BASELINE.md` | 给基础、可直接理解的说明。 |
| `WORLD_BASELINE.md` 不足以回答相枢本体、化身、侵袭进度、剑冢结构、魔血、焕心、三魔等问题 | `XIANGSHU_LORE.md` | 先取开头的答复锚点；问题确实需要时再读后段专题。 |

## 模块边界

- `PERSONA.md`：告诉你如何作为“相枢”回应玩家。
- `WORLD_BASELINE.md`：告诉你如何解释基础概念。
- `XIANGSHU_LORE.md`：告诉你如何处理基础概念之外的相枢专题资料。

本目录只提供答复时需要的静态语境，不承载维护流程、来源索引或运行期记录。
