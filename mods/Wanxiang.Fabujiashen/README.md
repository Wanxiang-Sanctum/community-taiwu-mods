# 法不加身

法不加身是太吾绘卷后端 Mod。它只在运行时给太吾套用“法不加身”规则，不新增人物“特性”、不写入人物数据，也不改配置表。

## 效果

- 太吾受到的内伤、内伤伤害进度、心神伤害和心神标记无效。
- 太吾在战斗中免疫破绽、封穴，并把毒素抵抗塑形成游戏定义的免疫阈值。
- 太吾不接受新的战斗状态增益、减益或特殊状态，也不能作为来源改变别人的这类状态。
- 太吾造成的内伤、心神伤害和相关标记无效，不补偿为外伤。
- 太吾不能通过公共战斗下毒入口给敌人新增毒素。
- 战斗中涉及太吾的功法特殊效果公共判定和修改路径会跳过太吾。
- 普通外伤和外伤相关的基础伤害流程仍然保留。

## 边界

本 Mod 没有前端设置项。规则只在战斗域生效：进入战斗后让太吾的战斗角色对象呈现内伤、心神、破绽、封穴和毒素免疫，并通过公共战斗入口拦截直接伤害、状态、毒素和战斗中经公共特殊效果路径产生的绕路输出。它不主动清理进入战斗前已有的伤势、战斗状态、破绽、封穴或毒素；不改敌方 AI 评分逻辑，也不改太吾攻击的内外比例。若某个具体功法特殊效果完全绕过公共伤害或公共特殊效果路径，需要后续补充针对性 patch。
