# 法不加身：一力破万法

法不加身是一款只作用于太吾的运行时规则 Mod。它不向存档写入真实人物特性，也不保存额外人物数据；人物特性列表中看到的“法不加身”只是前端标记，用来提示规则正在生效。

万法纷至，皆不加身。内伤、心神、毒素、破绽、封穴与诸般战斗异状，不再缠住太吾；但普通外伤仍是普通外伤，刀剑拳脚照样见真章。

## 效果

- 太吾免受新增内伤、内伤进度、心神伤害和心神标记影响。
- 太吾在战斗中获得内伤、心神、破绽、封穴和毒素免疫，新增战斗状态不会落到太吾身上。
- 太吾造成的内伤、心神、新增毒素和战斗状态不会生效。
- 涉及太吾的施展、入战类功法特殊效果会被跳过；装备、突破被动、真气、功法配置和普通外伤等基础流程保留。
- 直接施加的致命伤、致命标记和标记转移不会作用于太吾或由太吾造成；普通外伤积重至深，仍可能转成致命伤。

## 作用方式

- 规则只在运行时模拟，不新增存档中的真实人物特性，也不保存额外人物数据；关闭或移除 Mod 后，不会留下额外特性。
- 人物特性列表中的“法不加身”只是提示标记，且只显示在太吾身上。
- 已经存在的伤势、毒素、破绽、封穴、致命伤进度或战斗状态，不会被主动清空。
- 无需设置，启用后即对太吾生效。
