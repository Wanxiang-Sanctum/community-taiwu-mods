# 法不加身

法不加身是太吾绘卷后端 Mod。它只在运行时给太吾套用“法不加身”规则，不新增、不写入真实人物“特性”设定，也不改配置表。

## 效果

- 太吾受到的内伤、内伤伤害进度、心神伤害和心神标记无效。
- 太吾在战斗中免疫破绽、封穴，并把毒素抵抗塑形成游戏定义的免疫阈值。
- 太吾不接受新的战斗状态增益、减益或特殊状态，也不能作为来源改变别人的这类状态。
- 太吾造成的内伤、心神伤害和相关标记无效，不补偿为外伤。
- 太吾不能通过公共战斗下毒入口给敌人新增毒素。
- 涉及太吾的功法特殊效果公共判定和修改路径会跳过太吾。
- 普通外伤和外伤相关的基础伤害流程仍然保留。

## 边界

本 Mod 没有前端设置项。它通过后端 Harmony patch 生效：战斗中会先复用游戏战斗态免疫和毒抗入口塑形太吾的阶段性战斗状态，再用公共入口补丁兜住直接伤害和绕路效果。战斗塑形不主动清理进入战斗前已有的伤势、战斗状态、破绽、封穴或毒素。本 Mod 不改敌方 AI 评分逻辑，也不改太吾攻击的内外比例。若某个具体功法特殊效果绕过公共伤害或公共修改路径，需要后续补充针对性 patch。
