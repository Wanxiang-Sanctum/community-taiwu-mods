# 法不加身

法不加身是太吾绘卷后端 Mod。它只在运行时给太吾套用“法不加身”规则，不新增人物“特性”、不保存额外人物数据，也不改配置表。

## 效果

- 太吾受到的内伤、内伤伤害进度、心神伤害和心神标记无效；战斗外新增内伤也会被拦截。
- 相关判定会把太吾视为内伤、心神、破绽和封穴免疫；进入战斗后同步塑形战斗角色。
- 太吾在战斗中把毒素抵抗塑形成游戏定义的免疫阈值；战斗外新增毒素写入会被拦截。
- 太吾不接受新的战斗状态增益、减益或特殊状态，也不能作为来源改变别人的这类状态。
- 太吾造成的内伤、心神伤害和相关标记无效，不补偿为外伤。
- 太吾不能通过公共战斗下毒入口给敌人新增毒素。
- 战斗中涉及太吾的功法特殊效果公共判定和修改路径会跳过太吾。
- 普通外伤和外伤相关的基础伤害流程仍然保留。

## 边界

本 Mod 没有前端设置项，不新增人物特性，不保存额外人物数据，也不改配置表。

进入战斗后，它让太吾的战斗角色对象呈现内伤、心神、破绽、封穴和毒素免疫，并通过公共战斗入口拦截直接伤害、状态、毒素和战斗中经公共特殊效果路径产生的绕路输出。战斗外只兜底真实人物的新增内伤、新增毒素写入和人物免疫判定。

既有伤势、战斗状态、破绽、封穴或毒素不会被主动清理。它不全局伪装太吾的中毒免疫或真实人物毒抗，不改敌方 AI 评分逻辑，也不改太吾攻击的内外比例。若某个具体功法特殊效果完全绕过公共伤害或公共特殊效果路径，需要后续补充针对性 patch。
