# 界青快剑-连击版


## 功能

- **无限连击**：每次施展界青快剑后，根据几率判定是否继续连击
- **初始几率 60%**，每次连击后递减 15%，**保底 20%**
- **福缘关联**：初始几率 = 60% + 福缘 / 2
- **释放进度奖励**：连击时释放进度随连击数增长，每多一连增加 10%，最高 99%

## 安装

将 Mod 目录放入 `The Scroll Of Taiwu/Mod/` 下，在游戏内 Mod 管理中启用即可。

## 依赖

- 后端插件：`Plugins/Backend/JQKJ.Combo.Backend.dll`
- 前端插件：`Plugins/JQKJ.Combo.Frontend.dll`
